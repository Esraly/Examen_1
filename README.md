# Examen\_1 - Ruleta de Casino en C#



### Descripción

Este proyecto es una simulación de una ruleta de casino desarrollada en C#.
El usuario inicia con un saldo inicial y puede realizar diferentes tipos de apuestas, obteniendo ganancias según el resultado de la ruleta.



### Funcionalidades

* Saldo inicial de 300

##### Tres tipos de apuestas:

* Número específico → paga x10

  * Color (rojo/negro) → paga x5
  * Par o impar → paga x2

##### Generación de resultados aleatorios (simulación de ruleta)

##### Sistema de estadísticas:

* Balance actual
* Número de tiradas realizadas
* Número más frecuente
* Número menos frecuente
* Cantidad de rojos y negros
* Cantidad de pares e impares



### Interfaz

El programa funciona mediante consola, donde el usuario interactúa a través de un menú seleccionando opciones por número.



### Tecnologías utilizadas

* C#
* .NET
